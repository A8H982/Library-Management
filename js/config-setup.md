# Setup supabase config
To setup supabase config details in this project to make this work, below are the keys structure.

> Create a js file with named `supabase-config.js`


Paste the below code in that `supabase-config.js` with valid details.
```js
window.GJ_SUPABASE_URL = "url goes here";
window.GJ_SUPABASE_ANON_KEY = "anon key here";

window.GJ_TURNSTILE_SITE_KEY = "";

```
